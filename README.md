# Lavalampen-Rechner
Erstes J.S.-Projekt: Diese Anwendung soll beim Lavalampen restaurieren helfen. Dabei insbesondere beim berechnen der Korrekten Mengen bei der Lampen-Wachsherstellung. Auch die Flüssigkeit die zum Start beim Befüllen der Lampe benötigt wird kann berechnet werden und bestimmt werden, welche Glühbirne und, ob eine Heizspirale verwendet werden sollte.
