# Lavalampen-Rechner
Erstes JavaScript-Projekt: 
Diese Anwendung soll beim Lavalampen restaurieren helfen. 
Insbesondere beim Berechnen der korrekten Mengen bei der Lampen-Wachs-Herstellung müssen die Mengen verschiedenen Zutaten stimmen. 
Auch die Flüssigkeit die zum Start beim Befüllen der Lampe benötigt wird kann berechnet werden und es kann bestimmt werden, welche Glühbirne und, ob eine Heizspirale verwendet werden sollte.
